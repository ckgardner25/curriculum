# Digital Multimeter Overview
Author: Christopher Gardner, B.E.E. Candidate 
<br> <br>
One of the most important tools electrical engineers is the multimeter (DMM).  This tool always has a voltmeter, amperage meter, and resistance meter; some DMM's have thermal imaging, continuity tester, diode tester, and more. In this guide we will explore each tool and how to use them.
